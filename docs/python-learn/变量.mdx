## 变量 Variable

### 变量，变化的量。
> 非常简单的理解，在python中使用变量非常简单，比如：

```python 
name = '章三'
```
在上面程序中，我们定义了一个<Highcolor color='yellow'>变量  </Highcolor>`name`

`name`此时代表的就是`章三`,也可以说他们两个现在是等价关系，`=`不读做等于，读作`赋值`


:::info
  * 只能包含数字`(0-9)`，字母`(a-z,A-Z)`，下划线`_`。
  * 只能以字母或者下划线开头。
  * 变量名之间不能包含空格比如`class name`是不合法的，可以用下划线`class_name`
  > 在python3中变量名还可以包含其他Unicode字符，例如中文字符，但是我们<Highcolor color='red'>不推荐使用中文</Highcolor>

:::

### 变量能做什么

变量的本质是，你存入的内容。比如上边`age = 18` 此时变量age就是一个数字。在数学上我们都知道数字可以加减，因此变量age也可以加减。
示例：
```python
age = 18

age = age - 1

print(age)

```

你可能注意到`age = age - 1`这句话作用就是将年龄-1后，重新`赋值`给`变量age`,如果没有赋值操作，你打印age还是原来的值。同样的`加法,乘法，除法`也是如此


以上都是变量的简单使用，大家可能注意到我的变量名都是对应的英语单词，这样大家一看就知道它的作用。希望大家在定义变量时也能做到有意义！😁
