### for循环

🔗 在[while循环](./while循环.mdx)中，我们使用了循环中的一种，现在我们使用另外一种`for循环`。
与while循环不同的是，for循环不需要条件，它需要循环的对象，也就是对谁循环


```python title='for循环'

# 常用的
for i in range(10):
  print(i)

# 实际使用python中可能会用到的
book_list = ['水浒传','三国演义','红楼梦','西游记']
for i in book_list:
  print(i)
```
