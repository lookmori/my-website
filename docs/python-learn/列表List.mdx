### 列表List

```python

# 列表名 = [数据，数据，数据，....]

# 列表名的命名规则和变量名一样

fruit = ['西瓜','苹果','香蕉','桃子']

```
### 列表操作

* 列表取值

> 列表中每一项都有一个索引（下标），默认从0开始，因此第一个数据的索引就是0

```python
fruit = ['西瓜','苹果','香蕉','桃子']
# 索引     0      1     2     3


# 拿到fruit列表中的第一个水果
print(fruit[0])
```



* 修改列表值
>  将fruit列表中的第二个水果改为 哈密瓜

> 刚才已经拿到了第一个水果，写法为： 列表名[索引]

>  我们可以将`列表名[索引]` 看为变量

```python
fruit[1] = '哈密瓜'
```



* 删除列表值

>  删除fruit列表第三项

```python
# del 列表项
del fruit[2]
```


* 添加列表值

>  向fruit列表添加'葡萄'

```python
# 列表名.append('添加内容')
fruit.append('葡萄')
```
