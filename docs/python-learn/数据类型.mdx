🔗 在变量章节中，我们使用到了`整数int`,`字符串string`。这里我们继续介绍其他python中的数据类型

### 字符串string
> 使用单引号或者双引号
```python
name = '章三'
class_name = "四年级1班"
```


### 列表
> 以中括号包裹,以逗号分隔
> 你可以将任何东西加入列表中

```python
# 列表中的内容都有对应的索引(下标)从0开始，依次加1，最后一位的索引也可以是-1
book_list = ['水浒传','三国演义','红楼梦','西游记']

# 假如我想拿到西游记，应该怎么写？以下两种方式都是可以的
book_list[3]
book_list[-1]

# 假如我想拿到红楼梦，应该怎么写？以下两种方式都是可以的
book_list[2]
book_list[-2]


other_lists = ['apple',1,(246,369),{'name':'张安','age':18}]

```

### 元组
> 以小括号包裹,以逗号分隔
```python
color = ('红','绿','蓝')
```


### 字典
> 以大括号包裹,以逗号分隔，格式为`键:值`
```python
phone = {'name':'张三','phone_num':13788888888}
```
